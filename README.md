# カクテル原価計算ツール

バー営業で使うための、ブラウザだけで動くカクテル原価計算ツールです。ログインは不要です。

## 使い方

1. `index.html` をブラウザで開きます。
2. 材料名、ボトル価格、容量ml、使用量mlを入力します。
3. 各材料の1杯あたり原価と、合計原価が自動で計算されます。
4. 販売価格を入力すると原価率が表示されます。
5. 目標原価率を入力すると推奨販売価格が表示されます。

## スマホや他の端末で使う方法

### まず試すだけの場合

同じWi-Fiにつながっている端末なら、作業しているPCから一時的に配信して使えます。

1. PCでこのフォルダを開きます。
2. ターミナルで次のコマンドを実行します。

```bash
python3 -m http.server 4173 --bind 0.0.0.0
```

3. PCのIPアドレスを調べます。
4. スマホのブラウザで `http://PCのIPアドレス:4173/` を開きます。

この方法は、PCの電源が入っていて、同じWi-Fiにつながっている間だけ使えます。

### いつでも使いたい場合

GitHub Pages、Netlify、Cloudflare Pagesなどにこのフォルダを置くと、スマホや他のPCからURLで開けます。サーバー処理は使っていないので、静的サイトとしてそのまま公開できます。

公開後はスマホのブラウザでURLを開き、共有メニューからホーム画面に追加できます。2回目以降はアプリのように起動できます。

### GitHub Pagesで公開する場合

GitHub Pagesを使うと、友人に送れるURLを無料で作れます。GitHub公式ドキュメントでも、GitHub Pagesはリポジトリ内の静的ファイルを公開できる機能として案内されています。

1. GitHubにログインします。
2. 右上の `+` から `New repository` を選びます。
3. リポジトリ名を `cocktail-cost-tool` などにします。
4. 友人にURLを送る場合は `Public` を選びます。
5. `Create repository` を押します。
6. 作成したリポジトリで `uploading an existing file` を選びます。
7. このフォルダの中身をアップロードします。`cocktail-cost-tool` フォルダそのものではなく、フォルダ内の `index.html`、`styles.css`、`script.js`、`assets` などをアップロードしてください。
8. `Commit changes` を押します。
9. `Settings` → `Pages` を開きます。
10. `Build and deployment` の `Source` で `Deploy from a branch` を選びます。
11. `Branch` は `main`、フォルダは `/root` を選び、`Save` を押します。
12. 数分後に `https://ユーザー名.github.io/cocktail-cost-tool/` のようなURLで公開されます。

公開URLが出たら、そのURLを友人に送れば使えます。

### Netlifyなどで公開する場合

一番かんたんな方法は、Netlify Dropなどの静的サイト公開サービスに、このフォルダ一式をアップロードする方法です。

1. `cocktail-cost-tool` フォルダをZIPにします。
2. 静的サイト公開サービスにZIP、またはフォルダ一式をアップロードします。
3. 発行されたURLを友人に送ります。

このツールはブラウザ内だけで計算します。入力した材料や価格は、CSV保存をしない限り外部に送信されません。

## CSV保存・読み込み

- `CSV保存` を押すと、現在の材料、販売価格、目標原価率をCSVファイルとして保存できます。
- `CSV読み込み` から保存済みCSVを選ぶと、内容を復元できます。
- CSVはExcelやNumbersでも開けるよう、文字化けしにくいUTF-8 BOM付きで保存します。
- 表計算ソフトでCSVを開いたときの意図しない数式実行を避けるため、危険になりやすい記号から始まる文字は保存時に保護します。

## セキュリティについて

- 入力した材料名、価格、容量、使用量は外部サーバーに送信されません。
- ログイン、データベース、アクセス解析は使っていません。
- GitHub Pages上ではHTTPSで配信されます。
- 公開URLなので、リンクを知っている人は誰でもツールを開けます。
- ツールのHTML/CSS/JavaScriptは公開リポジトリ上で誰でも見られます。
- 入力データはサイト側に保存されません。CSV保存した場合のみ、保存した人の端末にファイルとして残ります。

## ファイル構成

- `index.html`: 画面の構造
- `styles.css`: バーらしい落ち着いたデザインとスマホ対応
- `script.js`: 原価計算、CSV保存、CSV読み込み
- `manifest.json`: スマホのホーム画面追加用設定
- `service-worker.js`: 公開環境でのオフライン再表示用キャッシュ
- `.nojekyll`: GitHub Pagesでそのまま静的ファイルとして公開するための設定
- `assets/bar-background.png`: 画面背景
- `assets/icon.svg`: アプリアイコン

## 計算式

- 材料原価 = ボトル価格 ÷ 容量ml × 使用量ml
- 合計原価 = 各材料原価の合計
- 原価率 = 合計原価 ÷ 販売価格 × 100
- 推奨販売価格 = 合計原価 ÷ (目標原価率 ÷ 100)
